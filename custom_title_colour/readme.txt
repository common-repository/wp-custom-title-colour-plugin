== Description ==

This WordPress plugin does exactly what its title suggests, 
gives you the ability to customise the title colour for any 
post via a panel in the sidebar of the write page. Handy for 
creating alerts, or drawing attention to a particular post.

== Installation ==

1. Upload the folder 'custom_title_colour ' into your 'wp-content/plugins' folder
2 .Activate the plugin via your plugins page in wordpress

== Usage ==

1 When writing or editing a post you can control the title colour via the 'Title Colour' 
tab in the right sidebar.
